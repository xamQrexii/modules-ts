# modules-ts

In this repostity we cover modules with:

- Array methods
- Interfaces
- If else
- Arrow function

- Arrow function with implicit return

- How data is stored in database

## How to run this project?

Clone this repository and run the following commands:
\
`tsc`
\
`node index.js`
